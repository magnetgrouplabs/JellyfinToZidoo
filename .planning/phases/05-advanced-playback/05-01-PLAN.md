  <files>app/src/main/java/com/jellyfintozidoo/JellyfinApi.java, app/src/test/java/com/jellyfintozidoo/IntroSkipperApiTest.java</files>
  <behavior>
    RED phase tests (IntroSkipperApiTest.java):
    - parseIntroSkipperResponse with both Introduction and Credits valid -> returns correct ms values for all 4 timestamps
    - parseIntroSkipperResponse with only Introduction (no Credits key) -> intro values set, credit values are -1
    - parseIntroSkipperResponse with only Credits (no Introduction key) -> credit values set, intro values are -1
    - parseIntroSkipperResponse with Introduction.Valid=false -> intro values are -1
    - parseIntroSkipperResponse with empty JSON "{}" -> all values are -1
    - parseIntroSkipperResponse converts seconds to milliseconds correctly (145.792s -> 145792ms)
    - IntroSkipperResult.introStartMs() returns -1 when introStartSec is -1
    - IntroSkipperResult.creditEndMs() converts correctly from seconds

    Implementation (JellyfinApi.java):
    - Add static inner class IntroSkipperResult with fields: introStartSec, introEndSec, creditStartSec, creditEndSec (all double, -1 = not present)
    - Add convenience methods: introStartMs(), introEndMs(), creditStartMs(), creditEndMs() that convert seconds to ms (return -1 if source is -1)
    - Add static method parseIntroSkipperResponse(String jsonBody) that parses the JSON using Gson JsonParser
    - Introduction object: read IntroStart/IntroEnd if Valid=true
    - Credits object: read IntroStart/IntroEnd if Valid=true (NOTE: Credits uses IntroStart/IntroEnd field names too, not OutroStart/OutroEnd)
    - Missing keys or Valid=false -> return -1 sentinels
  </behavior>
  <implementation>Follow existing parseItemResponse pattern: package-private static method, Gson JsonParser, inner result class with final fields.</implementation>
</feature>

<feature>
  <name>Feature 2: MediaStream Index Mapping and URL Param Extraction</name>
  <files>app/src/main/java/com/jellyfintozidoo/JellyfinApi.java, app/src/test/java/com/jellyfintozidoo/MediaStreamParsingTest.java</files>
  <behavior>
    RED phase tests (MediaStreamParsingTest.java):

    jellyfinToZidooAudioIndex tests:
    - [Video(0), Audio(1), Audio(2), Sub(3)] with jellyfinIndex=1 -> returns 0 (first audio)
    - [Video(0), Audio(1), Audio(2), Sub(3)] with jellyfinIndex=2 -> returns 1 (second audio)
    - [Video(0), Audio(1)] with jellyfinIndex=0 -> returns -1 (video stream, not audio)
    - [Video(0), Audio(1)] with jellyfinIndex=5 -> returns -1 (not found)
    - Empty array -> returns -1

    jellyfinToZidooSubtitleIndex tests:
    - [Video(0), Audio(1), Sub(2), Sub(3)] with jellyfinIndex=2 -> returns 1 (first sub, 1-based)
    - [Video(0), Audio(1), Sub(2), Sub(3)] with jellyfinIndex=3 -> returns 2 (second sub)
    - [Video(0), Audio(1), Sub(2)] with jellyfinIndex=1 -> returns -1 (audio, not subtitle)
    - Empty array -> returns -1

    parseUrlParam tests:
    - URL with AudioStreamIndex=2 -> returns 2
    - URL with SubtitleStreamIndex=3 -> returns 3
    - URL without the param -> returns -1
    - Malformed URL -> returns -1

    findDefaultStreamIndex tests:
    - MediaStreams with one IsDefault=true Audio -> returns that stream's Jellyfin Index
    - MediaStreams with no default Audio -> returns -1
    - MediaStreams with IsForced=true Subtitle -> returns that stream's Index
    - No matching type -> returns -1

    Implementation (JellyfinApi.java):
    - Add static int jellyfinToZidooAudioIndex(JsonArray mediaStreams, int jellyfinIndex): iterate streams, count Audio-type streams, when stream.Index == jellyfinIndex return audioCount (0-based)
    - Add static int jellyfinToZidooSubtitleIndex(JsonArray mediaStreams, int jellyfinIndex): iterate streams, count Subtitle-type streams, when stream.Index == jellyfinIndex return subtitleCount + 1 (1-based, 0=off)
    - Add static int parseUrlParam(String url, String paramName): parse URL, extract query parameter, return int or -1
    - Add static int findDefaultStreamIndex(JsonArray mediaStreams, String type): find first stream with IsDefault=true (or IsForced=true for subtitles) of given type, return its Index field
  </behavior>
  <implementation>All methods are package-private static for direct test access. Use Gson JsonArray/JsonObject for MediaStreams. Use android.net.Uri for URL parsing (or java.net.URI if android.net.Uri unavailable in unit tests -- check existing test pattern). For parseUrlParam, use simple string parsing to avoid Android dependency in unit tests.</implementation>
